### Vehicle
Para criar um vehicle  utilize o payload abaixo, como exemplo:

```json
{
  "plate": "AUO4429",
  "fleet": "11106110",
  "preventiveMaintenanceLimite": 6,
  "lastMaintenance": "Mon Jul 05 2021 19:55:48 GMT-0300 (Brasilia Standard Time)",
  "situation": "regular",
  "vehicleTypeId": ""
}
```
